Hologram pedestrians

How are holograms used?

A real car learns to avoid hologram pedestrians (zombies in this case). The interactions are recorded to train RL algorithms for avoiding pedestrians.

Tools used

The holograms were created with the Microsoft Hololens in Unity. The holograms communicate through ROS with the car to create a global reference frame.

When was this done?

This project was done during my freshman year with MIT's Aerospace Controls Laboratory
