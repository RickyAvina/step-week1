What did I learn

I learned Java and computer vision techniques with OpenCV with the help of some great mentors! I also learned how work in a team, under quite some stress.

Why computer vision?

In every FRC game, there is a 15 second autonomous period where the robot manipulates game pieces. Usually, there is a visual guide (retro-reflective tape) to help the robot "see".
